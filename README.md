# Updog API #

### Route: localhost:5000 ###

### EndPoints ###
### Posts ###


### Authentication ###

__/auth/register__  
post - register a user  

__/auth/login__  
post - login a user  

__/auth/logout__  
post - log out a user  

__/posts__  
_get - get all posts_  
post - create post  

__/posts/id__  
get - get post by id  
delete - delete post by id  
patch - update post by id
