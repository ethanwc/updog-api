# Updog API #

### Route: localhost:5000 ###

### EndPoints ###
### Posts ###


### Authentication ###

__/auth__  
_/auth/register_  
post - register a user  
_/auth/login_  
post - login a user
_/auth/logout_  
post - log out a user  

__/posts__  
_get - get all posts_  
post - create post  

__/posts/id__  
get - get post by id  
delete - delete post by id  
patch - update post by id
